# Resume Generation Optimization Project

**Goal:** Reduce token usage by 70% (25k → 7.5k tokens per resume)
**Timeline:** 3-4 weeks
**Cost Savings:** $0.20 → $0.06 per resume

```bash
cd optimization-project/

# Phase 1 (Week 1): Quick wins
./phase-1/implement.sh

# Phase 2 (Week 2-3): Memory + Knowledge Graph
./phase-2/implement.sh

# Phase 3 (Week 4): Learning system
./phase-3/implement.sh
```
